# UBUNSTONE
Un gestionnaire simple de serveur minecraft bedrock pour serveurs ubuntu

# installation

1.installation des modules nécéssaires:

-python 3.6+ (normalement installé par défaut)
-screen (avec la commande sudo apt install screen)
-curl (avec la commande sudo apt install curl)
-(OPTIONNEL) requests (avec la commande pip3 install requests)

2.modification du code:

-changer 
"SERVEUR_DOSSIER    = "CHEMIN DU DOSSIER DE VOTRE SERVEUR"

3.lancer le multitool:

lancez le multitool avec la commande 
"Python3 "chemin de votre fichier/Ubustone.py""
